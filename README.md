# bishop
[![UnitTest](https://github.com/clashmar/bishop/actions/workflows/unit-test.yml/badge.svg)](https://github.com/clashmar/bishop/actions/workflows/unit-test.yml)
[![MacOS](https://github.com/clashmar/bishop/actions/workflows/build-mac.yml/badge.svg)](https://github.com/clashmar/bishop/actions/workflows/build-mac.yml)
[![Windows](https://github.com/clashmar/bishop/actions/workflows/build-windows.yml/badge.svg)](https://github.com/clashmar/bishop/actions/workflows/build-windows.yml)

Bishop is a simple cross-platform 2D game editor for Windows and MacOS built with Rust and Lua. 

Please note that the current feature set is minimal and not yet suitable for production‑grade game development.
